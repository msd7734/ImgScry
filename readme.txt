usage: ImgScry [options] set_abbreviation
options:
 -f, --format <string>	 Format output filenames with C-style string formatting.
 -q, --quality <string>	 Image download quality (small, normal, large). Default is normal
 -s, --skip <true/false> Attempt to skip files that have already been downloaded. Default is true
 -l, --limit <number>	 Do not download more than this number of cards. Default is no limit
